# Introduction
Propeople is a comprehensive HR management system designed to streamline employee administration and workplace operations within an organization. It provides a central platform for managing key HR functions, allowing users to efficiently handle workforce-related tasks.

Through Propeople, HR teams and administrators can access and manage:

* Employee profiles and personal details
* Leave requests and approval workflows
* Department and role assignments
* Attendance and basic workforce tracking
* Internal user management and permissions

Propeople serves as both an operational tool and an organizational support system, helping companies improve efficiency, reduce manual HR processes, and maintain accurate employee records. Whether used by HR staff or managers, it simplifies day-to-day workforce management and ensures smoother communication and decision-making across the organization.

# User stories
documentation for user stories can be found [here](/documentation/user_stories/user_stories.pdf)

- ### As a user

* As a user, I want to log into the application, so that I can access my HR account securely.
* As a user, I want to request leave, so that I can apply for time off when needed.
* As a user, I want to view all my leave requests, so that I can track their status (approved or pending).
* As a user, I want to delete or modify my leave requests, so that I can correct mistakes or update my plans.
* As a user, I want to update my personal details, so that my information stays accurate and up to date.

⸻

### As an Admin (HR Manager)

* As an admin, I want to log into the application, so that I can manage the HR system.
* As an admin, I want to create new users, so that employees can be granted access to the system.
* As an admin, I want to delete users, so that inactive or irrelevant accounts are removed.
* As an admin, I want to view all users, so that I can keep track of employee records.
* As an admin, I want to view all leave requests, so that I can monitor and manage employee leave.
* As an admin, I want to approve or decline leave requests, so that I can manage staffing and workflow.
* As an admin, I want to update user details, so that employee records remain correct and current.
